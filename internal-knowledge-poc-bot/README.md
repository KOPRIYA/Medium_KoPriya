## Internal Knowledge POC Bot

A lightweight RAG-based internal knowledge assistant.

### Why this matters

- Demonstrates POC ownership
- Converts tribal knowledge into searchable intelligence
- Delivers real business value

### How to run

1. Add OpenAI API key
2. Install dependencies
   pip install -r requirements.txt
3. Ingest docs
   python ingest.py
4. Query knowledge
   python app.py
